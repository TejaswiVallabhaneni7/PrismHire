## 📦 Installation

```bash
git clone https://github.com/TejaswiVallabhaneni7/PrismHire.git
cd PrismHire
npm install
```

## 🚀 Quick Start

```bash
npm start
```
Then open (https://probable-rotary-phone-6vg7r6x99q536r9-5000.app.github.dev/) to view PrismHire in your browser.
